3.indexOf(String str)：获取str在字符串对象中第一次出现的索引

4.substring(int start)：从start开始截取字符串

6.startWith(String prefix)：判断字符串对象是否以指定的字符开头

10.trim()：取出字符串两端空格

11.split()：去除字符串中指定的字符，然后返回一个新的字符串
